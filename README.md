# Cybersecurity Awareness Site

A simple educational website that teaches common cybersecurity threats and good digital safety habits.

## What this project includes

- Informative pages for cyber threats, best practices, quiz, resources, and contact
- A dark/light theme toggle
- Responsive layout for desktop and mobile screens
- Interactive content for learning and self-check quizzes

## Project structure

- index.html — home page
- threats.html — common cyber threats overview
- best-practices.html — security best practices
- quiz.html — quiz page
- resources.html — helpful learning resources
- contact.html — contact page
- styles.css — site styling
- script.js — theme toggle, navigation, and interactive behavior

## How to view locally

1. Open the project folder in your browser.
2. Double-click index.html, or use a simple local server if you prefer.

Example with Python:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Notes

This project is built with plain HTML, CSS, and JavaScript, so no build step is required.
